# 标签词表

## 命名规则

| 规则 | 说明 | 示例 |
| --- | --- | --- |
| 品牌/官方大小写 | 遵循官方写法 | `iPhone`, `macOS`, `YouTube`, `ChatGPT` |
| 技术缩写大写 | — | `AI`, `API`, `SQL`, `LLM`, `NLP`, `CSS` |
| 中文通用词小写拼音或中文 | 根据可读性选择 | `读书`, `写作`, `心理学` |
| 无空格 | 多词用连字符 | `prompt-engineering`, `知识管理` |
| 不用同义重复 | 一个概念只保留一个标签 | `AI` ✅ / `人工智能` ❌ |
| 嵌套用 `/` | 层级不超过 2 层 | `编程/Python`, `设计/UI` |

### 新增标签流程

1. 检查本清单是否已有同义标签
2. 如果没有，确认该标签会被 3 篇以上笔记使用
3. 选择合适的类别添加到清单中
4. 遵循命名规则
